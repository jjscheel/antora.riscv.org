Presents a standardized control interface for RISC-V trace infrastructure (such as trace encoders, trace funnels, trace sinks) for the [Efficient Trace for RISC-V specification](./efficient-trace) and for the [RISC-V N-Trace (Nexus-based Trace)](./n-trace) specification. Standardized control interface allows trace control software development tools to be used interchangeably with any RISC-V device implementing processor and/or data trace.
