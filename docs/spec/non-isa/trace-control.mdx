# RISC-V Trace Control Interface

Presents a standardized control interface for RISC-V trace infrastructure (such as trace encoders, trace funnels, trace sinks) for the [Efficient Trace for RISC-V](https://riscv.atlassian.net/wiki/spaces/HOME/pages/809926657) specification and for the [RISC-V N-Trace (Nexus-based Trace)](https://riscv.atlassian.net/wiki/spaces/HOME/pages/809926691) specification. Standardized control interface allows trace control software development tools to be used interchangeably with any RISC-V device implementing processor and/or data trace.

*Latest version: 1.0    Date: November 2024*
